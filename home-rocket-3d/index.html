Homing Rocket Games

Seven versions, every one playable on its own. Open any file in a browser — no install needed.

1Homing Rockets 3D — Pure Pursuit
Fly a single target with the mouse; rockets chase it down using pure-pursuit guidance.
 2Many Targets
Up to 20 wandering targets; each rocket locks its nearest one and re-acquires when it dies.
 3100 Targets
Swarm mode — up to 200 targets, with a Full salvo button that fires one rocket per survivor.
 4Rocket Tank
Drive a tank with WASD; twin auto-tracking rocket pods fire homing missiles.
 5Minigun Tank
Six-barrel rotary launcher — hold fire to spin the barrels up to full RPM.
 6Minigun Tank — 2D Look
The same game rendered to look flat and cartoonish: toon shading, ink outlines, paper background.
 7Flying Nuke Tank
Your tank flies; rain homing nukes on an enemy tank army with mushroom-cloud blasts.

Each game is a single self-contained HTML file. They load the three.js library from a CDN, so an internet connection is needed the first time you open one. Keyboard controls are listed at the top of each game.
